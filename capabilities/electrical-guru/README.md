# Electrical Guru

> NEC 2023 code consultant for Denver Metro / Douglas County, CO.

## What It Does

Electrical Guru provides National Electrical Code (2023 edition) lookups and interpretation scoped to Denver Metro and Douglas County, Colorado jurisdictional requirements. It answers code questions, cites specific NEC articles with full section references, and flags local amendments relevant to Phoenix Electric's service area. Every response undergoes adversarial self-validation before delivery.
